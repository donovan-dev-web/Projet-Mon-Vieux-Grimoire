# Projet-Mon-Vieux-Grimoire
il s’agit d’un client qui souhaite ouvrir un site de notation de livres au nom de sa chaîne de librairies “Le Vieux Grimoire” à Lille. Le site s’appellera “Mon Vieux Grimoire”, et permettra aux membres d’ajouter un nouveau livre et de mettre une note visible par le public.
